# GenBox v2.5.0 Release Candidate

This rolling file points to the prepared version-specific release notes:

- [中文发布说明](release-notes-v2.5.0-zh.md)
- [English release notes](release-notes-v2.5.0.md)

The v2.5.0 release candidate introduces the Extension Center, bilingual
onboarding, unified workspaces and page headings, credential-vault controls,
GHCR-backed Docker Compose delivery, locked build dependencies, packaged-client
startup smoke tests, and SHA-256 release checksums.

Versioned release notes are frozen after the corresponding GitHub Release is
published. Ongoing implementation details belong in `docs/STATUS.md`; future
version-level changes belong under `Unreleased` in `CHANGELOG.md`.
